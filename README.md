# AI-VOICE-ASSISTANT-USING-OPEN-AI-AND-PYTHON
This project is an AI-powered voice assistant built using Python and OpenAI APIs. It enables users to interact with their system using natural voice commands, making tasks faster and hands-free. The assistant listens to user input through a microphone, processes the speech into text, and generates intelligent responses using OpenAI’s language model.

Features
Speech-to-text conversion
Intelligent responses using OpenAI
Text-to-speech output
Task automation (search, open apps, answer queries)
User-friendly and extensible design
Tech Stack
Python
OpenAI API
SpeechRecognition
pyttsx3 / gTTS
How It Works
Captures voice input via microphone
Converts speech to text
Sends query to OpenAI API
Processes response
Converts response to speech output
